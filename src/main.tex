\section{Mandatory tools}




\begin{textbox}{\href{https://ohmyz.sh/}{oh-my-zsh}}

\underline{\textbf{Plugins}}
\begin{itemize}
    \item \textbf{git} (autocompletion and explanation for git commands)
    \item \textbf{sudo} ( \keystroke{esc} + \keystroke{esc} toggle sudo before command)
    \item \textbf{docker} (autocompletion for docker)
    \item \href{https://github.com/ohmyzsh/ohmyzsh/wiki/Plugins}{Search for plugins for the tools you use the most} 
\end{itemize}

\underline{\textbf{Aliases}}
\begin{itemize}
    \item \textbf{$\cdot\cdot(\cdot\cdot)$ } (each dot appending '$\cdot\cdot$' means one folder before)
    \item \textbf{take <dir>} (mkdir <dir> ; cd <dir>)
    \item \textbf{d, 1, 2, 3...} (d shows lastest directories visited, each number moves to corresponding dir)
    \item \textbf{l} = ls -lah (list \textbf{l}ong, \textbf{a}ll and for \textbf{h}umans)
\end{itemize}

\green{Pros}

Very powerful \textbf{autocompletion} (\underline{case insensitive},  \underline{navigate} through autocompletion suggestions with \underline{arrow keys})\\

With \keystroke{tab} you can move/write args/see cmd's options quickly and easily\\

Start writing a cmd and press \keystroke{$\uparrow$} \keystroke{$\downarrow$} to navigate through latest cmds matching the written beginning \\

\href{https://github.com/romkatv/powerlevel10k}{Themes are really useful}. Provide information as: git status of current dir (if working directory is a repository), information about last execution return code, information of the time elapsed since command execution until termination, etc. 

\red{Cons}

Special matching char '$*$' does not work by default. (Fix it with \textit{echo 'setopt +o nomatch' >> \textasciitilde/.zshrc})\\

Special attention when doing \textit{\textbf{source} any-file\textbf{.bash}} files. Now will be \textit{\textbf{source} any-file\textbf{.zsh}}. (Attention using ROS)\\


Some scripts won't work with zsh, if strange errors occurs, just try using bash.

\begin{textbox}{terminator}




\green{Pros}

Multiple terminals in the same window with easy and quick navigation/creation/modification shortcuts. Highly recommended while working with ROS.

\yellow{Tips}

Right click on the terminal > Preferences > Profiles.
\\\\
Then in \textbf{scrolling} tab:
Add one more zero to scrollback lines number. Big errors contains more 500 lines
\\\\
Add one new \textbf{profile}, then go to \textbf{colors} tab, and select \textit{black on \textbf{white}} as built-in scheme. This profile is now available right clicking in the terminal over the profiles list. It is really needed when working at the sun.
\\

\section{Useful stuff}

\begin{textbox}{Shell shortcuts}

\keystroke{ctrl} $+$ \keystroke{a} $\rightarrow$ move cursor to the beginning of the line

\keystroke{ctrl} $+$ \keystroke{e} $\rightarrow$ move cursor to the end of the line

\keystroke{ctrl} $+$ \keystroke{l} $\rightarrow$ clear terminal 

\keystroke{ctrl} $+$ \keystroke{z} $\rightarrow$ suspend current process 

If a process is suspended, it can be resumed typing \textbf{fg} (foreground) or \textbf{bg} (background)


\keystroke{ctrl} $+$  \keystroke{$\leftarrow$}\keystroke{$\rightarrow$} $\rightarrow$ move cursor one word forward or backwards

\keystroke{ctrl} $+$ \keystroke{w} $\rightarrow$ cut last word

\keystroke{ctrl} $+$ \keystroke{k} $\rightarrow$ cut command from cursor to the end

\keystroke{ctrl} $+$ \keystroke{y} $\rightarrow$ paste cut string

\keystroke{alt} $+$ \keystroke{t} $\rightarrow$ swap last words

\keystroke{alt} $+$ \keystroke{.} $\rightarrow$ type last argument of last command

\end{textbox}

\begin{textbox}{Useful commands}

\mycommand{man}{an interface to the on-line reference manuals}

\mycommand{whatis}{ display one-line manual page descriptions}

\mycommand{whereis}{ locate the binary, source, and manual page files for a command}

\mycommand{which}{ locate a command}

\mycommand{whoami}{ print your effective userid}

\mycommand{watch}{  execute a program periodically, showing output fullscreen}

\mycommand{pwd}{ print working directory}

\mycommand{ping}{ send ICMP ECHO\_REQUEST to network hosts}

\mycommand{nmap}{ Network exploration tool and security / port scanner (\textit{sudo apt install nmap})}

\mycommand{ifconfig}{ multiple configuration for network interfaces (\textit{sudo apt install net-tools})}

\mycommand{htop}{ interactive process viewer (\textit{sudo apt install htop})}

\mycommand{head,tail}{  output the first/last part of files}

\mycommand{more,less}{ file perusal filter for crt viewing/the opposite}

\mycommand{history}{ print last executed commands}

\mycommand{find}{ search for files or dirs in a directory hierarchy}

\mycommand{grep}{ print lines matching a pattern}

\mycommand{neofetch}{ a fast, highly customizable system info script (\textit{sudo apt install neofetch})}

\mycommand{apropos}{ tells you commands that approximate the input you give}

\mycommand{gdb}{ the GNU debugger}

\mycommand{cmake-gui, ccmake}{ Graphical/Terminal User Interface for cmake (\textit{sudo apt install ccmake cmake-qt-gui})}

\mycommand{mount/umount}{Attach/unattach filesystem to specific mountpoint}

\mycommand{dmesg}{Print or control the kernel ring buffer}

\end{textbox}

\textbf{</>}	Input/Output redirect

\textbf{/}	Pathname directory separator

\end{multicols}
